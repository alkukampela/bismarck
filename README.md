# Let's play cards!

This is a multiplayer (3 or 4 players) card game.

## Local dev environment

### Prerequirements

* Node
* Docker & docker-compose


```sh
docker-compose up

npm install
npm run build
npm run start:watch

cd src/bismarck-web
npm install
npm run develop
```

Browse to http://localhost:1234/create
Fill the form and fetch tokens from terminal's stdout.
